La nivel de Homework, aplicatia a fost structurata profesional folosind Command Pattern, ceea ce inseamna ca fiecare actiune (salvare, vizualizare, listare) este un obiect separat, usor de intretinut. O componenta cheie este generarea de rapoarte prin FreeMarker, unde datele din Java sunt transformate automat intr-o pagina HTML organizata, oferind utilizatorului o interfata vizuala clara pentru consultarea catalogului direct in browser.

In sectiunea Advanced, catalogul a devenit unul "inteligent" prin adaugarea conceptelor (cuvinte-cheie) pentru fiecare resursa si implementarea unui algoritm Greedy. Acest algoritm rezolva problema matematica Set Cover, reusind sa selecteze cel mai mic numar de carti necesare pentru a acoperi o lista de materii dorite. Performanta sistemului a fost validata printr-un test de stres pe mii de inregistrari generate aleatoriu, demonstrand ca solutia optima este gasita aproape instantaneu.
